## 开发规范

### 提交规范

```
类型: 简短描述

详细说明（如果需要）

关联: ISS-XXX
```

类型包括：
- `fix`: 修复问题
- `feat`: 新功能
- `docs`: 文档更新
- `refactor`: 代码重构
- `test`: 测试相关
- `perf`: 性能优化

### 分支策略

- `main`: 稳定版本
- `develop`: 开发分支
- `feature/xxx`: 功能分支
- `fix/xxx`: 修复分支
