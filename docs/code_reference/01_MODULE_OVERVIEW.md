# TEXACO Python版本 - 模块划分与架构概述

## 1. 程序总体架构

TEXACO气化炉CFD模拟程序采用模块化设计，主要分为以下几个层次：

```
┌─────────────────────────────────────────────────────────────┐
│                        主程序层                              │
│                    src/main.py                               │
│  - 迭代控制                                                   │
│  - 收敛判断                                                   │
│  - 结果输出                                                   │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                       子程序层                               │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │initialization│  │gasifier_main│  │output_results│         │
│  │  (初始化)    │  │ (矩阵组装)  │  │  (结果输出)  │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
│  ┌─────────────┐                                            │
│  │  mass_flow  │                                            │
│  │ (质量流计算)│                                            │
│  └─────────────┘                                            │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                       函数计算层                             │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │gas_reactions│  │reaction_rates│  │  math_utils │         │
│  │ (气相反应)  │  │ (碳反应速率) │  │ (数学工具)  │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
│  ┌─────────────┐  ┌─────────────┐                           │
│  │fh2o_fortran │  │  common_data │                           │
│  │ (水蒸气物性)│  │ (全局数据结构)│                           │
│  └─────────────┘  └─────────────┘                           │
└─────────────────────────────────────────────────────────────┘
```

## 2. 模块详细说明

### 2.1 主程序模块 (main.py)

**文件位置**: `src/main.py`  
**对应Fortran**: `Wg1.for` (PROGRAM TEXACO_GASIFIER)

**主要功能**:
- 程序入口点
- 初始化调用
- 主迭代循环控制
- 收敛判断和残差计算
- 结果输出

**核心函数**:

| 函数名 | 输入参数 | 输出 | 功能描述 |
|--------|----------|------|----------|
| `main()` | 无 | 无 | 主程序入口 |
| `calculate_residuals()` | 无 | sumfe, sumwe, sumx, sumt | 计算当前迭代残差 |
| `print_matrices_iter0()` | 无 | 无 | 打印第一次迭代的矩阵 |

---

### 2.2 全局数据模块 (common/common_data.py)

**文件位置**: `src/common/common_data.py`  
**对应Fortran**: `COMMON.00`, `COMMON.01`, `COMMON.02`, `COMMON.03`

**主要功能**:
- 存储全局共享数据
- 实现Fortran Common块的Python版本
- 单例模式管理全局状态

**核心类**:

| 类名 | 功能 | 对应Fortran |
|------|------|-------------|
| `CommonData` | 全局数据容器 | Common块 |

**数据分类**:

```python
# Common.00 - 基本参数
NGAS = 8              # 气体组分数
G = 9.801             # 重力加速度 [m/s²]
RAG = 8314.3          # 通用气体常数 [J/(kmol·K)]
PI = 3.1415926        # 圆周率
P0 = 1.01325e5        # 标准大气压 [Pa]

# Common.01 - 几何参数
H[31]           # 格子高度位置 [m]
DELZ[31]        # 格子高度 [m]
AT[31]          # 格子截面积 [m²]
AR[31]          # 格子半径 [m]

# Common.02 - 求解变量
FEMF[9,31]      # 各组分摩尔流量 [kmol/s]
X[31]           # 碳转化率
T[31]           # 温度 [K]
WE[31]          # 碳质量流量 [kg/s]

# Common.03 - 矩阵
AMAT[13,13,32]  # 下对角块矩阵
BMAT[13,13,32]  # 对角块矩阵
CMAT[13,13,32]  # 上对角块矩阵
DMAT[13,32]     # 右端向量
```

---

### 2.3 初始化模块 (subroutines/initialization.py)

**文件位置**: `src/subroutines/initialization.py`  
**对应Fortran**: `Wg2.for`

**主要功能**:
- 读取输入数据文件 (Datain0.dat)
- 计算几何结构
- 初始化求解变量
- 计算进料参数

**核心函数**:

| 函数名 | 输入参数 | 输出 | 功能描述 |
|--------|----------|------|----------|
| `eingab(data_path)` | data_path: 数据文件路径 | 无 | 主初始化函数 |
| `geometry()` | 无 | 无 | 计算气化炉几何结构 |
| `qhcrct()` | 无 | 无 | 热量校正计算 |
| `_read_start_file(filepath)` | filepath: START.DAT路径 | 无 | 读取初始条件 |

---

### 2.4 气化炉主计算模块 (subroutines/gasifier_main.py)

**文件位置**: `src/subroutines/gasifier_main.py`  
**对应Fortran**: `Wg3.for` (SUBROUTINE GASIFIER)

**主要功能**:
- 构建块三对角矩阵系统的系数矩阵
- 计算每个网格单元的质量平衡和能量平衡
- 数值微分计算Jacobian矩阵

**核心函数**:

| 函数名 | 输入参数 | 输出 | 功能描述 |
|--------|----------|------|----------|
| `gasifier(...)` | 多个回调函数 | sum_ri | 主计算函数，构建矩阵 |
| `_calculate_dmat_for_cell(...)` | i, 回调函数 | rri | 计算单个单元的DMAT |
| `apply_variable_scaling()` | 无 | 无 | 变量缩放 |
| `restore_variable_scaling()` | 无 | 无 | 恢复缩放 |

---

### 2.5 质量流模块 (subroutines/mass_flow.py)

**文件位置**: `src/subroutines/mass_flow.py`  
**对应Fortran**: `Wg1.for` (SUBROUTINE XMASS)

**主要功能**:
- 计算每个网格单元的固体质量
- 计算停留时间
- 计算颗粒速度
- 计算空隙率

**核心函数**:

| 函数名 | 输入参数 | 输出 | 功能描述 |
|--------|----------|------|----------|
| `xmass()` | 无 | 无 | 计算质量流参数 |

---

### 2.6 气相反应模块 (functions/gas_reactions.py)

**文件位置**: `src/functions/gas_reactions.py`  
**对应Fortran**: `Wg6.for`

**主要功能**:
- 计算气相反应速率 (XK1-XK6)
- 计算挥发分释放 (FLUCHT)

**核心函数**:

| 函数名 | 输入参数 | 输出 | 功能描述 | 对应反应 |
|--------|----------|------|----------|----------|
| `xk1(i)` | i: 格子索引 | 反应速率 [kmol/s] | CO氧化反应 | CO + 1/2 O₂ → CO₂ |
| `xk2(i)` | i: 格子索引 | 反应速率 [kmol/s] | H₂氧化反应 | H₂ + 1/2 O₂ → H₂O |
| `xk3(i)` | i: 格子索引 | 反应速率 [kmol/s] | 水煤气变换 | CO + H₂O → CO₂ + H₂ |
| `xk4(i)` | i: 格子索引 | 反应速率 [kmol/s] | 逆水煤气变换 | CO₂ + H₂ → CO + H₂O |
| `xk5(i)` | i: 格子索引 | 反应速率 [kmol/s] | 蒸汽重整 | CH₄ + H₂O → CO + 3H₂ |
| `xk6(i)` | i: 格子索引 | 反应速率 [kmol/s] | CH₄氧化 | CH₄ + 2O₂ → CO₂ + 2H₂O |
| `flucht()` | 无 | 无 | 挥发分释放计算 | - |

---

### 2.7 碳反应速率模块 (functions/reaction_rates.py)

**文件位置**: `src/functions/reaction_rates.py`  
**对应Fortran**: `Wg7.for`, `Wg8.for`

**主要功能**:
- 计算碳与各种气体的反应速率 (A1-A5)
- 计算扩散系数 (XKC系列)
- 计算颗粒温度和结构参数

**核心函数**:

| 函数名 | 输入参数 | 输出 | 功能描述 | 对应反应 |
|--------|----------|------|----------|----------|
| `A1(i)` | i: 格子索引 | 反应速率 [kmol/s] | 碳-水蒸气反应 | C + H₂O → CO + H₂ |
| `A2(i)` | i: 格子索引 | 反应速率 [kmol/s] | 甲烷化反应 | C + 2H₂ → CH₄ |
| `A3(i)` | i: 格子索引 | 反应速率 [kmol/s] | 燃烧反应 | C + O₂ → CO/CO₂ |
| `A4(i)` | i: 格子索引 | 反应速率 [kmol/s] | Boudouard反应 | C + CO₂ → 2CO |
| `A5(i)` | i: 格子索引 | 反应速率 [kmol/s] | 催化水煤气变换 | CO + H₂O ⇌ CO₂ + H₂ |
| `RI(i)` | i: 格子索引 | 反应速率 [kg/s] | 总碳消耗速率 | - |
| `PHI(i)` | i: 格子索引 | 结构参数 | 机械因子计算 | - |
| `TPAR(i)` | i: 格子索引 | 温度 [K] | 颗粒温度计算 | - |
| `XKC_O2(i)` | i: 格子索引 | 速率系数 | O₂扩散系数 | - |
| `XKC_H2O(i)` | i: 格子索引 | 速率系数 | H₂O扩散系数 | - |
| `XKC_CO2(i)` | i: 格子索引 | 速率系数 | CO₂扩散系数 | - |
| `XKC_H2(i)` | i: 格子索引 | 速率系数 | H₂扩散系数 | - |

---

### 2.8 数学工具模块 (functions/math_utils.py)

**文件位置**: `src/functions/math_utils.py`  
**对应Fortran**: `Wg9.for`

**主要功能**:
- 块三对角矩阵求解 (BLKTRD)
- 矩阵运算 (加、减、乘、除)
- 高斯消元法
- 牛顿迭代和收敛判断

**核心函数**:

| 函数名 | 输入参数 | 输出 | 功能描述 |
|--------|----------|------|----------|
| `blktrd(nmat, nst)` | nmat: 矩阵大小, nst: 格子数 | 错误代码 | 块三对角矩阵求解 |
| `newtra(omega)` | omega: 松弛因子 | 无 | 牛顿迭代 |
| `kolon1(omega)` | omega: 松弛因子 | 无 | 更新变量，检查收敛 |
| `gausll(n, ns, a)` | n: 维数, ns: 右端列数, a: 增广矩阵 | nfehl, a | 高斯消元法 |
| `matadd(...)` | 矩阵参数 | matout | 矩阵加法 |
| `matsub(...)` | 矩阵参数 | matout | 矩阵减法 |
| `matmult(...)` | 矩阵参数 | matout | 矩阵乘法 |
| `matdiv(...)` | 矩阵参数 | matdiv, matout | 矩阵除法 |
